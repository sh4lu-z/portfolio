PastPapers.wiki Search API

GET /api/pastpapers.wiki

This endpoint allows you to search for past papers and marking schemes from the PastPapers.wiki database using a search query.

Request Parameters

Parameter Type Required Description
q String Yes The keyword to search for (e.g., "Physics", "2025 AL").

Success Response Example

{
  "creator": "Sh4lu Z",
  "status": "success",
  "search_query": "Combine Maths",
  "results_count": 4,
  "data": [
    {
      "title": "2025 A/L Combined Maths Marking Scheme",
      "pdf_link": "https://example.com/file.pdf"
    }
  ]
}

System Health Check

GET /api/health

This endpoint is used exclusively to check if the system is up and running properly.
