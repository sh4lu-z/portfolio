PastPapers.wiki Search API

Search for past papers and marking schemes dynamically from our main database.

GET /api/pastpapers.wiki

Query Parameters

ParamTypeRequiredDescription
qStringYesKeyword to search (e.g., "Physics").

Response Example

{
  "status": "success",
  "search_query": "Combine Maths",
  "data": [
    {
      "title": "2025 A/L Combined Maths",
      "pdf_link": "https://example.com/file.pdf"
    }
  ]
}
