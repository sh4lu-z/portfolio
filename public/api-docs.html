PastPapers.wiki Search API

GET /api/pastpapers.wiki

This endpoint allows you to search for past papers and marking schemes from the PastPapers.wiki database using a search query.

Request Parameters

Parameter Type Required Description
q String Yes The keyword to search for (e.g., "Physics", "2025 AL").

Response Fields

Field Type Description
creatorStringThe API developer's name.
statusStringReturns "success" if the request is handled.
search_queryStringThe query you searched for.
results_countIntegerTotal number of items found.
dataArrayList of objects containing title, image, pdf_link, and source.

Error Messages

If the q parameter is missing or formatted incorrectly, the API returns an error:

Condition Status Code Response Example
Missing q parameter 400 / 200 {"error": "?q=maths)"}

Success Response Example

{
  "creator": "Sh4lu Z",
  "status": "success",
  "search_query": "Combine Maths",
  "results_count": 4,
  "data": [
    {
      "title": "2025 A/L Combined Maths Marking Scheme | Sinhala Medium",
      "image": "https://pastpapers.wiki/wp-content/uploads/2026/03/2024-AL-Markings-1-350x250.jpg",
      "pdf_link": "https://pastpapers.wiki/download/39724/2025-a-l-marking/105220/2025-al-combined-maths-marking-sm-past-papers-wiki.pdf",
      "source_page": "https://pastpapers.wiki/2025-a-l-combined-maths-marking-scheme-sinhala-medium/"
    }
  ]
}
